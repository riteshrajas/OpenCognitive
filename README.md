# OpenCognitive — Free Matrix Reasoning Assessment

OpenCognitive is a transparent, 100% free matrix reasoning and fluid intelligence assessment based on open-source psychometrics. Designed with a premium, Apple-inspired minimalist aesthetic, the application runs entirely inside the user's browser, securing absolute privacy while generating high-fidelity diagnostic reports.

---

## 🧠 Psychometric Foundations & Science

The assessment battery measures **fluid intelligence (Gf)** and **visuospatial rotation** using standardized, open-access cognitive frameworks:

1. **Sandia Matrices:** Incorporates procedural item layouts featuring object relations, logic disjunctions (AND, OR, XOR), and coordinate transformations.
2. **ICAR (International Cognitive Ability Resource):** Utilizes public-domain validated cognitive test items designed for reliable unproctored online testing.
3. **MaRs-IB (Matrix Reasoning Item Bank):** Calibrated reasoning bank structure suitable for adult and adolescent cognitive evaluation.

### 📊 Mathematical Scoring Model
Scoring is calculated using **Item Response Theory (IRT)** with **Bayesian Expected A Posteriori (EAP)** ability estimation:

- Uses a 3-Parameter Logistic (3PL) model incorporating item difficulty ($b_i$), discrimination ($a_i$), and guessing floor ($c_i$):
  $$P_i(\theta) = c_i + \frac{1 - c_i}{1 + e^{-a_i(\theta - b_i)}}$$
- Approximates the latent trait score ($\theta$) using Gauss-Hermite quadrature over 41 nodes spanning $[-4, 4]$:
  $$\hat{\theta}_{EAP} = \frac{\sum_{k=1}^{41} q_k L(\mathbf{u}|\theta_k) P(\theta_k)}{\sum_{k=1}^{41} L(\mathbf{u}|\theta_k) P(\theta_k)}$$
- Converts latent ability ($\theta$) to a standard IQ metric ($M=100, SD=15$) for user readability.

## 🔒 Client-Side Privacy
Unlike typical online psychometric tests, OpenCognitive preserves absolute privacy:
* All IRT calculations run locally in the browser's JavaScript environment.
* Responses are never sent to external servers or databases.
* PDF report generation is compiled on the client side using `jspdf` and downloaded directly.

## 🛠️ Local Development

OpenCognitive is built with **Next.js (App Router)**, **React**, **TypeScript**, and **Tailwind CSS**.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/riteshrajas/OpenCognitive.git
   cd OpenCognitive
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the local development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser to see the app.

4. Build for production:
   ```bash
   npm run build
   ```

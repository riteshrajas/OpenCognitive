# Psychometric Framework

OpenCognitive evaluates **fluid intelligence ($G_f$)**—the capacity to think logically and solve problems in novel situations, independent of acquired knowledge.

The assessment utilizes two primary public-domain item design standards:

---

## 🧩 1. Sandia Matrices
The Sandia Matrices framework provides procedurally structured, nonverbal reasoning puzzles. These items require identifying visual rules across a $3 \times 3$ grid where the bottom-right cell is missing.

OpenCognitive matrices incorporate the following procedural logic operations:
* **Object Relations:** Changes in color, shading, orientation, or size of shapes across columns or rows.
* **Logical Disjunctions:** Combining shapes using boolean-like operations:
  * **AND:** Elements must be present in both parent shapes to appear in the child.
  * **OR:** Elements from either parent shape appear in the child.
  * **XOR:** Elements appearing in only one parent shape appear in the child (elements in both are cancelled out).
* **Coordinate Transformations:** Systematic rotation or reflection of coordinate planes across cells.

---

## 📐 2. ICAR (International Cognitive Ability Resource)
The ICAR project provides open-access, validated measures of cognitive abilities. OpenCognitive draws inspiration and validation from ICAR item styles:

* **Three-Dimensional Spatial Rotation:** Mental rotation tasks requiring the user to identify matching cube rotations or folded net layouts.
* **Letter-Number Series:** Inductive reasoning items requiring users to deduce alphanumeric patterns (e.g., progression gaps, alternating sequences).
* **Verbal Syllogisms:** Deductive reasoning challenges assessing logical flow and syllogistic correctness from arbitrary premises.

# Welcome to the OpenCognitive Wiki!

OpenCognitive is an open-source, client-side psychometric evaluation tool designed to measure **Fluid Intelligence ($G_f$)** and **Visuospatial Rotation** using validated, public-domain scientific instruments.

This wiki provides comprehensive documentation of the project's scientific frameworks, math models, and local code architecture.

## 🚀 Key Architectural Pillars
1. **100% In-Browser Computation:** Test scoring is completed locally inside the user's browser, eliminating data tracking and safeguarding privacy.
2. **Apple-Inspired Minimalism:** The UI prioritizes visual focus and readability, replacing dense laboratory workbench layouts with spacious, high-contrast typography.
3. **Open Science Framework Compliant:** Standardized item parameters and transparent documentation permit academic scrutiny and research use.
